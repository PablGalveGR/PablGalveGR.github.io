Proyect: Library management web application

Made with Django Framework

This proyect contains a web aplication wich allows the users to manage a library.
Some functionalities are:

  • Guest/Client/Adminitrarion:
    • See all the books and filter the search results
    • See all the book's instances and filter the search results
    • See all the Authors and filter the search results
  • Administration:
    • Add/Delete books.
    • Add/Edit/Borrow/Return/Delete book instances.
    • Add/Delete authors.
    • Add/Delete users.
  • Client:
    • Borrow books.
